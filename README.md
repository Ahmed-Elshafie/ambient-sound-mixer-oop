# Ambient Sound Mixer (OOP)

This project is a JavaScript-based Ambient Sound Mixer built using Object-Oriented Programming (OOP) principles.

## 📌 Project Overview

The application allows users to play and mix different ambient sounds with individual volume controls, creating a relaxing and customizable audio experience.

## 🧠 Technologies Used

* HTML5
* CSS3
* JavaScript (OOP)

## 🚀 Features

* Play multiple ambient sounds
* Individual volume control for each sound
* Clean and modular OOP structure
* Simple and interactive UI

## 📁 Project Structure

This project is built on a template and extended with JavaScript functionality using multiple JS files for better organization.

## 📦 How to Run

1. Clone the repository:

```bash
git clone https://github.com/username/ambient-sound-mixer-oop.git
```

2. Open `index.html` in your browser

## 🎯 Purpose

This project is part of learning JavaScript OOP and improving frontend development skills.
